# Kajak Verleih

## Requirements

Install `php` and `composer`.

## Development

### As Docker

```
docker-compose up
```

Then call `localhost:8080`.

### Via PHP

```shell
php -S 127.0.0.1:8080
```

Then call `localhost:8080`.

### Via Composer

```shell
composer start
```

Then call `localhost:8080`.
